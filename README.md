<h4 align="center">基于vue前后端分离的快速开发框架</h4>

#### 模块说明

```
daidaiqi-ui
└── public
     ├── img-----------------图片
└── src
     ├── api-----------------全局请求
     ├── common--------------全局配置
     ├── components----------全局组件
     ├── hooks---------------hooks
     ├── http----------------全局请求封装
     ├── layout--------------页面基本布局
     ├── mock----------------数据包
     ├── router--------------路由
     ├── store---------------状态
     ├── style---------------样式
     ├── utils---------------全局方法
     └── views---------------页面
```
